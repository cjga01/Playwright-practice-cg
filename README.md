# Playwright Automation Practice

A robust End-to-End (E2E) automated testing repository built for learning and practicing modern QA automation strategies using Microsoft Playwright.

---

## 🚀 Tech Stack & Tools

* **Core Framework:** [Playwright Test](https://playwright.dev/) (`v1.61.0`)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Type-safe bindings with `@types/node v25.9.3`)
* **Test Runner:** Native Playwright Runner
* **Reporting:** Built-in Playwright HTML Reporter

### 🎯 Target Application
The automation suites in this repository are designed to test real-world user flows on the following mock e-commerce platform:
* **Target Website:** [QA Brains eCommerce Practice](https://practice.qabrains.com/ecommerce)

---

## 📋 Prerequisites

Before running this project locally, make sure you have the following installed:
* [Node.js](https://nodejs.org/) (LTS version recommended)
* [Git](https://git-scm.com/)
